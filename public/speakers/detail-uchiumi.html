グローバル展開のコネクティッドカーを支える大規模サーバーレスシステム事例

セッション概要

今回開発した自動車（モビリティ）向けの「コネクティッドサービス」は、AWS Fargate や AWS Lambda を中心とした数百のマイクロサービスで構成されており、それらを AWSの複数リージョン上に展開していますが、

  • プラットフォームアーキテクチャの全体像
  • AWS Fargate や AWS Lambda 等のサーバーレスサービスを選択した理由
  • サーバーレス・アーキテクチャーで構築する過程で発生した技術課題や性能課題およびその対処方法
  • 開発や運用において注意すべき内容

プラットフォームアーキテクチャの全体像 AWS Fargate や AWS Lambda 等のサーバーレスサービスを選択した理由 サーバーレス・アーキテクチャーで構築する過程で発生した技術課題や性能課題およびその対処方法 開発や運用において注意すべき内容 といった点について、紹介したいと考えています。自動車という高信頼性を要求されるサービスに対してこれほどの大規模なプラットフォームを構築する過程で経験した課題・獲得した知見は、今後サーバーレスがより普及していく中でエンタープライズでの極めて優れた参考事例になると考えています。
